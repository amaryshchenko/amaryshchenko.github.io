# Classified Intelligence Dossier
**An interactive, terminal-inspired cybersecurity portfolio.**

## Overview
This repository contains the source code for my professional cybersecurity portfolio, stylized as a highly restricted, classified intelligence dossier. It is designed to be fully interactive and immersive, moving away from traditional resume formats to demonstrate a hands-on, hacker-oriented aesthetic.

## Features
- **Biometric Security Simulation:** A CSS/JS-driven fingerprint scanner that requires sustained haptic interaction to unlock restricted documents (CV and certificates).
- **Ultraviolet (UV) Investigation Mode:** A hidden toggle that shifts the color palette into an ultraviolet spectrum, revealing "invisible ink" handwritten notes from a simulated intelligence director.
- **Threat Intelligence Ticker:** A scrolling banner mimicking live CISA and US-CERT alerts.
- **Purge Protocol:** A hidden terminal command that visually wipes the dossier from the DOM.
- **Terminal Aesthetics:** Custom CRT scanlines, brutalist typography, and monochromatic design elements inspired by legacy mainframe systems.

## Tech Stack
- **HTML5:** Semantic structure and layout.
- **CSS3:** Custom variables, CSS animations (glitch effects, progress bars, scanlines), and interactive states.
- **Vanilla JavaScript (ES6+):** DOM manipulation, interval-based effects, and asynchronous haptic logic without relying on heavy frameworks.
